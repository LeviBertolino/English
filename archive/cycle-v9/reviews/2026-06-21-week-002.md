## 1. Recap — One-Line Summary per Unit

| Unit | Title | Core rule |
|---|---|---|
| **009** | I have… and I've got… | *have* and *have got* both express possession; *have got* only for states, not habits; pick one system per sentence |
| **010** | was/were | Past of *be*: I/he/she/it → *was*; you/we/they → *were*; *you* is always *were*, even for one person |
| **011** | worked / got / went (Simple Past affirmative) | Completed past actions: regular verbs add *-ed*; irregular verbs change form and must be memorised |
| **012** | I didn't… / Did you…? (Simple Past negative/questions) | *did/didn't* + **base form** always — the main verb never goes into past form after *did/didn't* |
| **013** | I was doing (Past Continuous) | *was/were + verb-ing* = action in progress at a past moment; the "rolling movie" vs. simple past "snapshot" |
| **014** | I was doing and I did (Past Continuous vs. Simple Past) | Past continuous = background in progress; simple past = event/interruption; *while* → background, *when* → event |

### Exercise 1 — Possession: choose *have* or *have got* (Unit 009)

Choose the correct form. In some gaps, only one is possible; in others, both work — mark with *(both)* if applicable.

> **IT onboarding form:**
>
> 1. *"\_\_\_ you a company laptop yet?"* (formal written)
> 2. *"Yes — I \_\_\_ it, but I \_\_\_ (not) VPN access yet."* (informal Slack)
> 3. *"\_\_\_ she a direct report on this team?"* (informal speech)
> 4. *"We \_\_\_ breakfast meetings every Friday."* (routine — only one form works here!)

✅ **Answers:**
1. **Do you have** (formal context — *Have you got* is also valid, mark *(both)* if you wrote that)
2. **I've got** it / **I haven't got** VPN access (contractions, informal Slack — *"I have it / I don't have"* also correct)
3. **Does she have** / **Has she got** *(both)*
4. **We have** — ✅ only *have* works here. *Have got* is for possession states, NOT routines or habits. *"We've got breakfast meetings every Friday"* ❌

### Exercise 4 — Simple past negative/questions: error correction (Units 011 + 012)

Fix the four errors. (All involve *did/didn't* + wrong verb form.)

1. *"Did the CI pipeline **ran** all the tests before the merge?"*
2. *"We **didn't shipped** the feature because the security review wasn't complete."*
3. *"Did she **reviewed** the pull request yesterday?"*
4. *"They **didn't found** the root cause in time."*

✅ **Answers:**
1. *"Did the CI pipeline **run** all the tests before the merge?"* (*did* → base form)
2. *"We **didn't ship** the feature…"* (*didn't* → base form)
3. *"Did she **review** the pull request yesterday?"* (*did* → base form)
4. *"They **didn't find** the root cause in time."* (*didn't* → base form)

**The pattern:** if *did* or *didn't* is in the sentence, the main verb is always the **base/infinitive form**. No exceptions.

### Exercise 6 — Past continuous vs. Simple past: fill in (Unit 014)

This exercise is based directly on the post-mortem mini-context from Unit 014. Complete with the correct form of the verb in brackets:

> *"Last Thursday, our team \_\_\_ (run) a load test on the new API gateway. While the test \_\_\_ (run), the monitoring dashboard \_\_\_ (show) normal metrics. Then, at exactly 11:43am, latency \_\_\_ (spike) to 8 seconds. I \_\_\_ (review) the logs when a junior engineer \_\_\_ (notice) the bottleneck — a single misconfigured timeout. We \_\_\_ (fix) it in under five minutes."*

✅ **Answer:**
*"Last Thursday, our team **ran** a load test. While the test **was running**, the dashboard **was showing** normal metrics. Then, latency **spiked**. I **was reviewing** the logs when a junior engineer **noticed** the bottleneck. We **fixed** it in under five minutes."*

**Why each:**
- *ran* = completed action (we did the test — event)
- *was running / was showing* = background state while the test was in progress
- *spiked* = event that interrupted the background
- *was reviewing* = in progress when the junior engineer noticed
- *noticed / fixed* = completed events in sequence

### Exercise 7 — Mixed tenses in context: choose the right form (All units)

Choose the correct option (A or B) for each sentence. Both options are grammatically formed — choose the one that fits the **meaning**:

1. *"I \_\_\_ the design doc when the product manager called for an urgent meeting."*
   - A) *wrote*
   - B) *was writing*

2. *"She \_\_\_ the database migration last Sunday night. It took about three hours."*
   - A) *ran*
   - B) *was running*

3. *"Did the deployment \_\_\_ the traffic spike during the load test?"*
   - A) *handled*
   - B) *handle*

4. *"The staging cluster \_\_\_ down during the test — but the production environment was fine."*
   - A) *wasn't*
   - B) *weren't*

5. *"Do you \_\_\_ access to the observability stack, or should I request it for you?"*
   - A) *have*
   - B) *have got*

✅ **Answers:**
1. **B** — *was writing* (background action in progress when the interruption happened)
2. **A** — *ran* (completed event — the migration is done; the whole event is described as finished)
3. **B** — *handle* (base form required after *did*)
4. **A** — *wasn't* (singular subject: *staging cluster*)
5. **A** — *have* is better in a formal written question; **B** (*have got*) is also grammatically correct — mark *(both)* if you chose B

### Exercise 8 — Translation challenge: Brazilian Portuguese → English (All units)

Translate these sentences. They contain common error traps for Portuguese speakers.

1. *"O sistema não tinha os recursos necessários."* (Use *have* system)
2. *"Você estava na reunião quando o alerta disparou?"* (past continuous question)
3. *"Ela não revisou o PR antes de fazer o merge."* (simple past negative)
4. *"Enquanto o time de backend refatorava a API, o time mobile já estava entregando uma nova versão do cliente."* (while + parallel past continuous)
5. *"O deploy foi um sucesso, mas o rollback não foi documentado direito."* (was/were)

✅ **Answers:**
1. *"The system **didn't have** the necessary resources."* (or *"The system **hadn't got**…"* — but *didn't have* is cleaner and standard)
2. *"**Were you** at the meeting when the alert fired?"* (NOT *"Was you"* — *you* always takes *were*)
3. *"She **didn't review** the PR before merging."* (NOT *"didn't reviewed"* — base form after *didn't*)
4. *"**While** the backend team **was refactoring** the API, the mobile team **was already shipping** a new client version."*
5. *"The deploy **was** a success, but the rollback **wasn't** documented properly."*

## 3. 📝 Integrative Writing Task (self-directed)

Write a short paragraph of **5–7 sentences** describing a real or hypothetical engineering incident, retrospective, or project delivery. Use **at least 3 of the 6 structures** from this week. Aim to use them naturally — don't force them all in.

**Target structures to draw from:**
- *have / have got* (possession/availability)
- *was/were* (past state of being)
- Simple past affirmative (completed events)
- *didn't / did…?* (past negative or question)
- Past continuous *was/were + -ing* (background action in progress)
- Past continuous + simple past together (background + interruption)

**Model paragraph:**

> *Last quarter, our team shipped the v2 data pipeline on time — but the rollout wasn't smooth. We didn't have a proper runbook for the migration step, and the team was still writing one when the deploy started. While the pipeline was processing the first batch of records, the database connection pool ran out — something nobody tested at that scale in staging. The on-call engineer was reviewing unrelated alerts when she noticed the spike. She fixed it in about eight minutes. The system now has better observability, and we've got a documented recovery plan for the next time this happens.*

**Structures used:**
- *shipped* (simple past affirmative)
- *wasn't* (was/were)
- *didn't have* (simple past negative)
- *was writing* (past continuous)
- *was processing* / *ran* (past continuous + simple past — interrupted action)
- *was reviewing* / *noticed* (past continuous + simple past)
- *has* / *we've got* (have/have got — present possession, for contrast)

## 4. 🇧🇷 Resumo em português — O que você consolidou, o que ainda precisa de atenção

### O que deve estar sólido agora

Esta semana consolidou um núcleo fundamental da gramática inglesa do passado. O **simple past** (afirmativo, negativo e interrogativo) é a estrutura que você mais vai usar em comunicação profissional — post-mortems, retrospectivas, atualizações de status, changelogs. A regra do *did/didn't + forma base* é simples e absoluta: se *did* ou *didn't* está presente, o verbo principal **nunca** vai para o passado. O **was/were** como passado de *be* deve estar automático, incluindo a armadilha de *you → always were*, mesmo para uma pessoa.

O **past continuous** (was/were + -ing) também deve estar familiar: é o "pano de fundo" do passado — o que estava em andamento quando o evento principal aconteceu. A combinação *past continuous + when + simple past* (ou *while + past continuous*) é uma estrutura de alta frequência em inglês técnico e você deve ser capaz de produzi-la com naturalidade agora.

### O que provavelmente ainda precisa de trabalho

O **have / have got** tem nuances que levam mais tempo: a restrição de *have got* (apenas posse, nunca hábitos), e os pares de questão/negativa (*do/does* vs. *have/has*). Se o Exercise 1 exigiu pausas, vale revisar a Unit 009 este fim de semana.

A distinção **past continuous vs. simple past** ainda pode gerar hesitação — especialmente decidir se uma ação é um "evento completo" (simple past) ou um "fundo em andamento" (past continuous). O Exercise 6 e o Exercise 7 são os melhores para voltar se isso ainda parecer instável.
