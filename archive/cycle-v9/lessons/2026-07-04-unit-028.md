# Unit 028 — Used to (do)

## 0. 🔁 Warm-up: Quick Recall (≈60 seconds)

Yesterday's lesson — **Unit 027: Reported speech (2): questions and requests** — covered reporting questions and orders: yes/no questions take **if/whether** (*"Are you happy?"* → *she asked if I was happy*), wh-questions keep the question word but drop inversion (*"How is the migration going?"* → *she asked how the migration was going*, NOT *how was it going*), and orders use **tell/ask + object + to-infinitive** (*"Close the ticket"* → *she told me to close the ticket*).

Three rapid recall questions:

1. Report this: *"Did you fix the bug?"* → She asked ___ I ___ the bug.
2. What's wrong with *"He asked where did I go"*?
3. How do you report *"Don't deploy on Friday"* (as a request from your manager)?

👉 **Answers:**
1. ✅ *"She asked **if/whether** I **had fixed** the bug."*
2. ✅ No inversion after the question word — *"He asked where **I went**."*
3. ✅ *"She **told me not to deploy** on Friday."*

## 2. 🧠 Adquirir — Mini-context first, rule second

> **Levi, catching up with an old colleague from a previous job:**
>
> **Colleague:** Still writing backend code every day?
>
> **Levi:** Not really — I **used to write** code all day, but now I spend most of my time in design reviews and 1:1s.
>
> **Colleague:** Ha, same. I **used to think** management would be less stressful than engineering. I was wrong.
>
> **Levi:** Right? We **used to ship** features on our own. Now it's all about unblocking other people.
>
> **Colleague:** Did the team **use to be** this big?
>
> **Levi:** No — it **didn't use to have** more than five engineers. Now we're twenty.

Notice:
- **used to + base verb** describes a past habit, repeated action, or state that is **no longer true now**.
- It only exists in the past — there's no present form. For a current habit, just use the present simple (*"I write code every day"*, not *"I use to write"*).
- Negative: **didn't use to** (no *-d* after "use" in negatives and questions).
- Question: **Did ... use to ...?**

### The rule

| Form | Example |
|---|---|
| Affirmative | I **used to work** in a small startup. |
| Negative | The company **didn't use to have** an AI team. |
| Question | **Did** you **use to code** every day? |
| Past state | She **used to be** the only backend engineer here. |

> ⚠️ Common trap: **"used to"** vs **simple past**. Both can describe past actions, but **used to** highlights that something was a *repeated habit or lasting state* which has since **changed**. *"I worked at Google"* is just a fact; *"I used to work at Google"* nudges toward *"...and now I don't."*
> ⚠️ Spelling trap: it's **use to** (no *-d*) after **did/didn't**, but **used to** in the plain affirmative. ❌ *"I didn't used to like it"* → ✅ *"I **didn't use to** like it."*
> ⚠️ Don't confuse with **"be used to"** (= be accustomed to something) — that's a different structure entirely, for another day.

## 3. ✏️ 8 example sentences

1. Our team **used to deploy** manually every Friday — now it's fully automated.
2. The startup **didn't use to have** a dedicated QA process before the Series A.
3. **Did** engineers **use to write** their own deployment scripts here?
4. Our CTO **used to be** an individual contributor before moving into leadership.
5. I **used to think** code reviews were just a formality.
6. She **used to live** in São Paulo before she moved for the job.
7. We **used to play** football every weekend, but now we barely find time.
8. My grandfather **used to tell** long stories after dinner.

## 4. 🛠️ Practice — recognition + production

**Recognition/gap-fill:**
Fill the gap: *"The team ___ (not / use to) run daily standups — we introduced them last year."*

✅ **Answer:** *"The team **didn't use to** run daily standups — we introduced them last year."*

**Production task:**
Write 2 sentences about how your engineering team or workflow used to be, using *used to*.

**Model answer:**
- *"We used to review every PR manually before we adopted automated linting."*
- *"The team didn't use to have an on-call rotation — everyone just reacted to alerts ad hoc."*

## 5. 💡 Memory tip

**"Used to" = a photo from the past that no longer matches today.** If you can mentally add *"...but not anymore"* after the sentence, *used to* fits. *"I used to code every day (but not anymore)."* If the habit is still true today, drop *used to* and use plain present simple instead.

## 8. 🇧🇷 Resumo em português

Hoje vimos **"used to + verbo no infinitivo sem "to""**, usado para descrever **hábitos ou estados do passado que não são mais verdade hoje** — por exemplo, *"I used to write code every day"* (eu costumava escrever código todos os dias, mas não faço mais isso). Essa estrutura só existe no passado: não existe "I use to" no presente para hábito atual — nesse caso usa-se o present simple normalmente.

Cuidado com a ortografia: no afirmativo é **used to** (com "-d"), mas nas negativas e perguntas com **did/didn't**, o verbo volta a ser **use to** (sem "-d") — *"I didn't use to like it"*, não *"I didn't used to like it"*. Evite também confundir com **"be used to"** (estar acostumado a algo), que é uma estrutura totalmente diferente e será vista em outra lição.
